## âš ï¸ Requirements

This package requires the following peer dependencies to be installed in your project for correct work:

- Vue 3: `vue@^3`
- Tailwind CSS 4: `tailwindcss@^4` or higher
- Lucide Vue Next: `lucide-vue-next@^0.526` or higher
- TW Animate CSS: `tw-animate-css@^1` or higher

Check the following guides to install required peer dependencies in your `Vue 3 project`:

- [Vue 3 Installation Guide](https://vuejs.org/guide/quick-start.html)
- [Tailwind CSS 4 Installation Guide](https://vuejs.org/guide/quick-start.html)
- [TW Animate CSS Installation Guide](https://www.npmjs.com/package/tailwindcss-animate)
- [Lucide Vue Installation Guide](https://lucide.dev/guide/packages/lucide-vue)

## ðŸ“¦ Package Installation

### Step 1: Configure GitHub Packages Access

> âš ï¸ **Important:**
> To install this package from Grapegram registry you will need to get a `personal access token` from your GitHub account with `read:packages` scope enabled. Then you will need to add it to your `.npmrc` file in your project root folder like this:
>
> Note that every time when you will need to update and reinstall packages, you will need to have this token set in your environment variable `GITHUB_TOKEN` or you can replace `${GITHUB_TOKEN}` with your actual token string but be careful not to push it to public repositories.

Create a `.npmrc` file in the root of your project:

```
@grapegram:registry=https://npm.pkg.github.com/
//npm.pkg.github.com/:_authToken=${GITHUB_TOKEN}
```

### Step 2: Install via npm or yarn

**Via npm:**

Latest verison:

```bash
npm install @grapegram/ui-kit
```

Specific version:

```bash
npm install @grapegram/ui-kit@0.2.3
```

**Via yarn:**

Latest version:

```bash
yarn add @grapegram/ui-kit
```

Specific version:

```bash
yarn add @grapegram/ui-kit@0.2.3
```

You can also `add the package` as a dependency into your `package.json` file manually:

```json
{
  "dependencies": {
    "@grapegram/ui-kit": "0.2.3"
  }
}
```

Then run `npm install` or `yarn` to install the package.

## ðŸŽ¨ Style Integration

### Basic Integration (components only)

> In that case you will use only components without design tokens. It means that your `tailwind` will use `its own default configuration` instead of Grapegram's tailwind configuration.

Import styles in your `main.ts`:

```typescript
import { createApp } from 'vue'
import App from './App.vue'
import '@grapegram/ui-kit/styles'

createApp(App).mount('#app')
```

### Advanced Integration (with design tokens)

If you want to use `Grapegram preconfigured tailwind` and `design tokens` in your components, create a styles file:

**src/styles/main.css:**

```css
@import 'tailwindcss';

@import '@grapegram/ui-kit/tailwind-config'; // Import Grapegram Tailwind configuration (provides tailwind utility classes)
@import '@grapegram/ui-kit/theme'; // Import light and dark theme variables (design tokens)

/* Your custom styles */
@layer base {
  * {
    @apply border-border outline-ring/35;
  }

  body {
    @apply bg-background text-foreground font-body;
  }
}
```

Import this file in `main.ts`:

```typescript
import './styles/main.css'
```

## ðŸŒ“ Dark Theme

The library supports dark theme out of the box. Add the `dark` class to the root element:

```tsx
<script setup lang="ts">
import { ref } from 'vue'

const isDark = ref(true)
</script>

<template>
  <div :class="{ dark: isDark }">
    <div class="bg-background text-foreground min-h-screen">
      <Button @click="isDark = !isDark"> Toggle Theme </Button>
      <!-- Your content -->
    </div>
  </div>
</template>
```

```tsx
<p class="font-body text-base">Body text</p>
<small class="text-muted-foreground text-xs">Caption</small>
```
